# HiAuRo 功能测试文档

## 测试准备

### 环境要求
- FFXIV 游戏客户端 + XIVLauncher（开发模式）
- Dalamud 已安装
- HiAuRo 插件已编译并放置到 DevPlugins 目录
- HiAuRo.FA 插件已编译并放置到 DevPlugins 目录

### 编译命令
```bash
# HiAuRo 主插件
dotnet build HiAuRo/HiAuRo.csproj -nologo

# HiAuRo.FA 场地分析插件
dotnet build HiAuRo.FA/HiAuRo.FA.csproj -nologo
```

## 已知问题

1. **HiAuRo.FA 构建**：WSL 环境下输出路径不存在，需要在 Windows 环境构建
2. **辅助轴路径**：当前辅助轴直接 ForceSpell→SlotExecutor，理想路径是向智能层提需求（设计目标，非 bug）
3. **伤害窗口合并**：尚未实现，当前逐事件独立计算减伤
4. **技能数据**：部分职业的 StatusID 可能需要根据游戏版本更新
