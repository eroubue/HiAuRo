# Phase 9: Authoring Layer — 网页三轴编辑器

## 概述

纯 HTML/CSS/JS 实现的三轴时间线编辑器，零外部依赖，Apple iOS 设计风格。本地文件操作，无需 C# 后端。

## 文件清单

| 文件 | 作用 |
|------|------|
| `HiAuRo/UI/web/editor.html` | 三栏布局：文件列表 + 编辑器主区 + 属性/快捷面板 |
| `HiAuRo/UI/web/editor.js` | 全部逻辑（~1000 行）：树编辑器、时间线、文件操作、属性面板 |
| `HiAuRo/UI/web/editor.css` | Apple iOS 风格设计系统 |
| `HiAuRo/Authoring/AuthoringServer.cs` | C# WebSocket 后端（备用，当前编辑器不依赖） |
| `doc/dev-tasks/PHASE9_AUTHORING.md` | 本文档 |

## 架构

```
┌────────────┬──────────────────────────────────┬──────────────┐
│  左侧      │           编辑器主区               │   右侧       │
│  文件列表   │  执行轴: 缩进树（彩色标签+子节点）  │  轴信息      │
│  选择目录   │  事实轴: 时间线（阶段轨道+事件条）  │  快捷添加    │
│  浏览文件   │  辅助轴: 同执行轴                  │  属性面板    │
│  点击加载   │                                   │              │
└────────────┴──────────────────────────────────┴──────────────┘
```

## 三种轴

| 轴 | 编辑器 | 数据模型 |
|----|--------|----------|
| 执行轴 | 手写缩进树 | `ExecutionTimelineData` / `TriggerNodeData` → 11 种节点类型 |
| 事实轴 | 手写 CSS 时间线 | `FactTimelineData` / `FactPhase` / `FactEvent` / 7 种动作类型 |
| 辅助轴 | 同执行轴 | 同执行轴 |

## 文件操作

全部使用 File System Access API（`showOpenFilePicker` / `showSaveFilePicker` / `showDirectoryPicker`），fallback 到 `<input type="file">` 和 Blob 下载。不需要任何后端。

| 操作 | 方式 |
|------|------|
| 新建 | 内存生成空模板 |
| 加载 | 文件选择器 + 文件拖拽 + 左侧目录浏览 |
| 保存 | 写回原文件 (FileHandle) / 另存为 |
| 导出 | Blob 下载 |
| 导入 | 文件选择器覆盖当前内容 |

## 执行轴功能

- **11 种节点类型**: Root、序列、并行、选择、循环、条件、动作、延迟、脚本、调试输出、清除等待
- **缩进树**: 彩色左边框 + 类型标签 + 名称，子节点缩进显示
- **快捷添加**: 右侧面板 8 个快捷按钮（序列/条件/动作/延迟/循环/并行/选择/脚本）
- **右键菜单**: 添加子节点（含类型选择）、上移/下移、删除
- **`＋` 按钮**: 点击弹出类型选择菜单
- **拖拽排序**: 同层级拖拽调整顺序
- **属性面板**: 名称/ID/启用/备注/标签，类型特定字段（条件/动作/延迟/循环/脚本）

## 事实轴功能

- **时间标尺**: 顶部 mm:ss 格式
- **阶段轨道**: 横向 lane，左侧有色标识
- **事件条**: 按时间定位的色块，显示名称
- **拖拽调时**: 按住事件条拖拽调整时间
- **分支**: 每个阶段 lane 有 `＋分支` 按钮，分支缩进显示
- **属性面板**: 事件名/ID/时间/持续，同步校准（startsUsing/ability/inCombat + 技能ID），7 种动作类型

### 动作类型

| 类型 | 字段 |
|------|------|
| demand | 减伤%, 治疗 |
| skillSuggestion | 技能ID, 名称, 优先级 (high/normal/optional) |
| setVariable | 变量名, 值 (true/false) |
| toggleVariable | 变量名 |
| logMessage | 消息 |
| switchPhase | 目标阶段, 标签 |
| switchBranch | 条件变量, 目标分支 |

## 设计系统

Apple iOS 风格：暗色背景、毛玻璃工具栏、SF 字体、大圆角卡片、微妙阴影、平滑动画。所有下拉组件为自定义 iOS 风格（`iosSelect`），代替原生 `<select>`。
