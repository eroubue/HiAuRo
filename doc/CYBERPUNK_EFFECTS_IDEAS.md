# 赛博朋克 ImGui 特效推荐

基于 ImDrawList 原生 API（AddLine/AddCircle/AddRect/AddBezierCubic/PathLineTo 等）的能力分析，
结合赛博朋克/霓虹风格视觉美学，推荐以下 5 个最适合 HiAuRo 窗口背景的特效方案。

---

## 1. 霓虹电路板 (Neon Circuit Board)

**描述：** PCB 走线风格的动态电路网络，发光节点之间有数据脉冲沿路径流动。

**实现思路：**
- 预生成一组水平和垂直走线（`AddLine`，霓虹青/品红色），交汇处放置圆形节点（`AddCircleFilled`）
- 走线用两层绘制：底层宽线（低 alpha）做辉光 + 顶层细线（高 alpha）做主体
- 数据脉冲：沿路径移动的高亮小圆点（`AddCircleFilled`），每条线段上 1-2 个脉冲，速度随机
- 节点呼吸闪烁：用 `EffectUtils.StatePulse()` 控制亮度
- 鼠标悬停时最近的节点放大发光

**互动性：** 鼠标靠近节点时触发脉冲加速扩散，点击节点沿连接线发射电磁脉冲波。

---

## 2. 雷达扫描仪 (Radar Scanner)

**描述：** 中心雷达扫描效果，旋转扇形扫过时检测并点亮"目标"光点，配合同心圆和十字准线。

**实现思路：**
- 同心圆（`AddCircle`，3-4 层，间距递增）+ 十字准线（`AddLine`）
- 旋转扫描扇形：用 `PathLineTo` + `PathArcTo` 构建扇形，`PathFillConvex` 填充半透明渐变
- 扫描线：从中心到边缘的一条发光射线（双层 AddLine 实现辉光）
- 目标点：随机分布在各环上的小圆点，被扫描线扫过时亮度渐亮再渐灭（基于角度差计算 alpha）
- 底层加淡色网格（复用 GeometricGlowEffect 的 DrawGrid 逻辑）

**互动性：** 鼠标位置作为"主目标"，显示距离和方位角标注文字（`AddText`）。点击切换"锁定跟踪"模式，被锁定目标高亮。

---

## 3. 数据流管道 (Data Flow Pipeline)

**描述：** 横向管道网络，彩色数据包沿管道流动，节点处有进出动画，类似赛博朋克电影中的数据可视化。

**实现思路：**
- 管道：用 `AddRectFilled` 绘制水平/垂直窄条（半透明深色作为管道壁）
- 管道内部用更亮的 `AddLine` 绘制中心流动指示线
- 数据包：沿管道移动的小矩形（`AddRectFilled`），颜色随"数据类型"变化（青、品红、黄）
- 节点：管道交叉处的圆形（`AddCircle` + `AddCircleFilled`），有呼吸发光效果
- 每个节点定期"发射"新数据包到随机连接的管道
- 整体做一个从左到右的缓慢滚动偏移，让管道看起来无限延伸

**互动性：** 鼠标点击管道节点，触发"数据爆发"——同时向所有连接管道发射一批发光数据包。鼠标悬停的管道段高亮加速。

## 4. 全息投影 (Holographic Wireframe)

**描述：** 半透明线框几何体缓慢旋转，带扫描线干扰和色差闪烁，像科幻电影中的全息显示屏。

**实现思路：**
- 在窗口中心区域绘制旋转线框多面体（复用 GeometricGlowEffect 的 DrawPolygon，增加 3D→2D 投影）
- 顶点用 `AddCircleFilled` 标注，棱用 `AddLine` 连接，整体低 alpha (0.15-0.3)
- 色差效果：同一线框偏移 ±2px 分别用青色和品红色绘制（模拟 RGB 分离）
- 扫描线干扰：水平方向随机位置的短横线（`AddLine`），alpha 随机闪烁
- 整体加入 2-3 秒周期的"闪烁故障"：偶尔整体 alpha 瞬间降到 0.05 再恢复
- 底层加淡蓝色渐变矩形（`AddRectFilled`）模拟全息投影底色

**互动性：** 鼠标移动控制投影旋转角度（模拟观察角度变化）。鼠标靠近几何体时边缘增亮（接近"触摸全息"效果）。

---

## 5. 故障艺术 (Glitch Art)

**描述：** 窗口边缘的故障撕裂效果 + 随机色彩条纹 + 文字噪点，赛博朋克 UI 的标志性视觉风格。

**实现思路：**
- 水平撕裂：在窗口上随机 Y 位置绘制水平位移条纹（将窗口某区域用 `AddRectFilled` 绘制半透明色块，相邻条纹水平偏移 2-8px）
- 色彩通道分离：对条纹区域分别用青、品红、黄三色叠加绘制
- 噪点带：随机位置绘制 1-3px 高的彩色水平线（`AddLine`，霓虹色）
- 闪烁定时器：每 3-5 秒触发一次"大故障"（持续 0.1-0.3 秒），期间故障密度翻倍
- 边缘锯齿线：用 `AddLine` 在窗口四边绘制不规则的锯齿波形（模拟信号干扰）
- 用 `ImGui.GetTime()` 控制故障的时序，大部分时间安静，偶尔突发

**互动性：** 鼠标快速移动时触发额外故障。点击窗口任何位置产生一次"电磁脉冲"故障波（从点击位置向外扩散的圆环干扰）。

## 可行性对比

| 特效 | 实现难度 | 性能开销 | 互动性 | 赛博感 |
|------|---------|---------|--------|--------|
| 霓虹电路板 | 中 | 低 | 高 | ★★★★★ |
| 雷达扫描仪 | 低 | 低 | 高 | ★★★★☆ |
| 数据流管道 | 中高 | 中 | 中 | ★★★★☆ |
| 全息投影 | 中 | 低 | 中 | ★★★★★ |
| 故障艺术 | 低 | 极低 | 高 | ★★★★★ |

**推荐优先实现顺序：** 故障艺术 > 雷达扫描仪 > 霓虹电路板 > 全息投影 > 数据流管道

理由：故障艺术实现最简单且赛博感最强；雷达扫描仪互动性最好；电路板视觉效果最丰富。

## ImDrawList 关键 API 参考

本项目已有的辉光技巧（双层绘制）：
- 辉光层：宽线条/大圆 + 低 alpha（0.05-0.15）
- 主体层：细线条/小圆 + 中等 alpha（0.3-0.6）

可用的 ImDrawList 原语：
- `AddLine` — 线条（走线、扫描线、撕裂线）
- `AddRect` / `AddRectFilled` — 矩形（管道、色块、数据包）
- `AddCircle` / `AddCircleFilled` — 圆（节点、目标点、光晕）
- `AddBezierCubic` — 贝塞尔曲线（曲线路径、波形）
- `PathLineTo` / `PathArcTo` / `PathStroke` / `PathFillConvex` — 自定义形状（扇形、多边形）
- `AddText` — 文字（数据标注、HUD 信息）
- `PushClipRect` / `PopClipRect` — 裁剪区域
